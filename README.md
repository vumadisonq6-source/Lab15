# Lab 15: Coffee Order Assistant
This program takes the user input, coffee order and membership status, and calculates the total then displays the receipt.

## How to Run
1. Open the program in your IDE.
2. Enter student names and scores.
3. View the calculated grade output.

##Example Output
=== Coffee Menu ===
1. Espresso ($3.50)
2. Latte ($4.50)
3. Cappuccino ($5.00)
Enter your choice (1-3): 3
Are you a cafe member (Y/N)? Y
--- Bill ---
Drink: Cappuccino
Base price: $5.0
Discount: 10.0%
Final price: $4.5
